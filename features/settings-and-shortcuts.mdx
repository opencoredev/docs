# Settings and Shortcuts

The Settings route (`/settings`) exposes five sections: Account, Providers, Chat, Models, and Shortcuts.

## Account

- Edit display name.
- View connected email/OAuth provider.
- Delete account through safeguarded workflow endpoint.

## Providers

- Select active provider for chat generation.
- Connect/disconnect provider credentials.
- See cloud usage state and limits.
- OpenRouter supports OAuth plus API key style flows.

## Chat

- Configure generated title style (short/standard/long).
- Enable/disable delete confirmations.

## Models

- Toggle provider filter label style.
- Refresh model cache.
- Inspect available model count and freshness.

## Shortcuts

Shortcut system is fully configurable and grouped by General, Navigation, and Chat categories.

Examples:

- Toggle sidebar
- Open settings
- New chat
- Focus prompt
- Stop generation

Reserved browser bindings are blocked to avoid conflicts.
